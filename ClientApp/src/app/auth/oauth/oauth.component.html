💡 About OAuth 2.0

  • Grant Types: Authorization Code, Implicit, Client Credentials, Password
  • Components: Resource Owner, Client, Authorization Server, Resource Server
  • Security: PKCE for public clients, state parameter for CSRF protection
  • Tokens: Access Token (API access), Refresh Token (renew access)
  • Use Cases: Third-party app access, SSO, API authentication
