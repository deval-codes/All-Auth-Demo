🔐 What is JWT?

JWT (JSON Web Token) is a compact, URL-safe token format used for securely transmitting information between parties as a JSON object.

💡 About JWT

  • Stateless: No server-side session storage needed
  • Portable: Can be used across different domains
  • Self-contained: Contains all user information
  • Secure: Signed to prevent tampering (use HTTPS!)
  • Storage: Store in httpOnly cookies or memory (avoid localStorage for sensitive data)
